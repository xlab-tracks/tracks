External sites can be quoted inline with the same card machinery: a
<SiteQuote
  href="https://example.com/"
  site="IANA"
  title="Example Domain"
  excerpt="This domain is for use in documentation examples without needing permission."

the real page on click (tap opens the card first on touch; the excerpt and
attribution are all inline props — no registry).
